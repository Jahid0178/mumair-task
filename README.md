## Run Locally

Clone the project

```bash
  git clone https://github.com/Jahid0178/mumair-task.git
```

Go to the project directory

```bash
  cd mumair-task
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Tech Stack

**Client:** React, Next, TailwindCSS, Swiper, React Icons, Typescript
